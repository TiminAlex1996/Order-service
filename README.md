# Тестовое задание Order-service

## Описание приложения

REST-сервис предоставляет интерфейс для сохранения заказов и вывода всех заказов.

Документацию по API см. [спецификацию](OrderServiceSpecification.yaml).

## Стэк технологий:

- Java 11;
- Maven;
- Spring Boot;
- Spring Web;
- Spring Data JPA;
- Liquibase;
- PostgreSQL;
- JUnit;
- Mockito;
- Docker;
- Swagger.
